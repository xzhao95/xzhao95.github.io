React源码阅读笔记-React API

发表于 2023-02-10 分类于 web前端 阅读次数： Valine：

当前react最新版本为18.2.0。从主包React中开始阅读，文件地址为：react/packages/react/src/React.js
可以从当前文件中看到，当前文件主要导出一些React API
React包中主要是对这些API的封装

React.Children

提供map、forEach、count、toArray、only方法，用于处理props.children，对children进行批量处理。
在forEach、count、toArray方法中，主要都使用了map方法

核心方法mapChildren：主要用于调用回调函数，递归调用mapIntoArray方法扁平化children，并赋予每一个child新key

React.Component

用于创建一个React组件

由setState和forceUpdate方法可见，主要是调用updater方法

React.PureComponent

继承自Component，并赋值了一个isPureReactComponent的标识

React.createElement

根据参数，用于创建节点，该方法中主要处理属性配置以及子节点，并将处理好的参数传进ReactElement方法中
