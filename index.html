博客搭建记录

发表于 2020-12-27 分类于 web前端

博客系统： Hexo
部署环境：github page

前言

前端接触有三年了，想搭建个属于自己的博客记录工作记录学习。通过网上的教程搭建了这个博客，记录一下搭建过程。

搭建前提

  1. node安装
  2. git安装
  3. github账号注册
  4. 重点：Hexo安装

hexo托管在国外，下载慢，可通过国内淘宝镜像进行安装

搭建正题

本地初始化

发布至github

  1. 需要在github上建一个仓库，仓库名需命名为 (用户名).github.io
  2. 配置本地文件
    找到_config.yml文件

    修改deploy部分
  3. 本地运行命令 如果提示 那就再安装一次发布插件

查看

路由：https://(用户名).github.io/
发布后需等待3-5分钟。
网上有看到一帖，说是仓库名命名为(用户名).github.com，不需要等待就能立即查看到更改，还未试验。

总结

搭建过程还是很简单哒，全程大部分时间都是安装等待，等待ing。。。
